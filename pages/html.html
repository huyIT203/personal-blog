HTML là gì?

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu tiêu chuẩn được sử dụng để tạo và thiết kế nội dung trên trang web.
Với HTML, bạn có thể tổ chức các phần tử như tiêu đề, đoạn văn, danh sách, hình ảnh, liên kết, và nhiều thành phần khác.

HTML thường được sử dụng để:

  • Định nghĩa cấu trúc cơ bản của trang web
  • Kết hợp với CSS để định dạng giao diện
  • Kết hợp với JavaScript để thêm tính năng động
  • Hỗ trợ SEO bằng cách tổ chức nội dung rõ ràng

Các thẻ HTML cơ bản

  • <h1>...</h1>: Thẻ tiêu đề
  • <p>...</p>: Thẻ đoạn văn
  • <a>...</a>: Thẻ liên kết
  • <img>: Thẻ hình ảnh
  • <ul>...<li>...</li></ul>: Thẻ danh sách
